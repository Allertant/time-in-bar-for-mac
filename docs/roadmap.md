# TimeInBar 迭代规划

## 已完成

### 基础

- 原生 macOS 菜单栏应用骨架（`MenuBarExtra`，`LSUIElement`）
- 固定时间段模式：开始/结束时间、刷新频率、三态切换、`剩余时间 · 进度`
- 起步项目文档（`docs/`、`CLAUDE.md`）

### 模式与计算

- 倒计时模式（`按时长`）：手动"开始上班"，按 0.5 小时步进配置工作时长
- 固定时间段支持夜班（`start >= end` 视为跨天）
- 倒计时支持跨午夜（会话 end 在今天或更晚就保持有效）

### 自动化与集成

- 登录后自动启动（`SMAppService`）
- 下班 1 分钟后自动退出（仅调度未来定时器，不补退）
- 上班时自动启动 / 下班时退出 Stretchly
- 下班到点全屏黑色提示（3 秒自动消失，ESC 可关）

### 显示与可访问性

- 进度展示：百分比 / 饼图，可分别开关剩余时间与进度
- 状态栏图标的 VoiceOver 标签

### 工程基础

- 迁移到 Xcode app 壳 + 本地 SPM 包（`TimeInBarKit`）架构
- `WorkScheduleCalculator` 纯函数化 + 23 个单元测试
- 本地化目录（zh-Hans 源语言，en 翻译）
