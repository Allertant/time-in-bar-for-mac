# TimeInBar 迭代规划

## P0 已完成

- 搭建原生 macOS 菜单栏应用骨架
- 支持开始时间、结束时间、刷新频率设置
- 支持开始前 / 工作中 / 下班后 三态切换
- 工作中展示 `剩余时间 · 进度`
- 加入基础项目文档

## P1 建议尽快补齐

- 打磨状态栏图标语义
- 首次启动默认体验优化
- 配置窗口视觉优化
- 增加应用图标资源
- 完整本地构建验证

## P2 可提升可用性

- 开机自动启动
- 更精确的定时刷新边界
- 更自然的文案和菜单层级
- 增加“当前设置时间段”的展示
- 增加异常配置的引导

## P3 可选扩展

- 工作日启用配置
- 提醒功能
- 多套预设模式
- 自定义文案
- 数据统计

## 当前建议

下一步优先级最高的不是继续堆功能，而是：

- 让工程在你的机器上直接稳定运行
- 看真实状态栏宽度是否舒服
- 观察 `3h32m · 23%` 在实际使用中的可读性
- 再决定是否调整文案或图标
